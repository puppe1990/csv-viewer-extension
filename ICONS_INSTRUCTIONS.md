# Instruções para Criar os Ícones

A extensão precisa de 3 ícones para funcionar corretamente. Você pode criar ícones de várias formas:

## Opção 1: Usar um Gerador Online

1. Acesse https://www.favicon-generator.org/ ou https://realfavicongenerator.net/
2. Faça upload de uma imagem ou crie um ícone simples
3. Baixe os tamanhos 16x16, 48x48 e 128x128
4. Renomeie os arquivos para:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`
5. Coloque os arquivos na pasta da extensão

## Opção 2: Criar Manualmente

Use qualquer editor de imagens (Photoshop, GIMP, Paint, etc.) para criar:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

Sugestão de design: Use um ícone de planilha/tabela ou CSV com cores roxas/azuis para combinar com o tema da extensão.

## Opção 3: Usar Ícones Temporários

Você pode usar qualquer imagem PNG e redimensionar para os tamanhos necessários usando ferramentas online como:
- https://www.iloveimg.com/resize-image
- https://imageresizer.com/

## Nota Importante

Sem os ícones, a extensão pode não funcionar corretamente. Certifique-se de criar os 3 arquivos antes de carregar a extensão no Chrome.
